🧠 Architecture Overview

This portfolio follows a content-driven structure, making updates fast and scalable without touching UI logic.

src/
├── components/     # UI components
├── pages/          # Section views
├── content/        # Profile, projects, skills data
├── hooks/          # Custom hooks
├── context/        # Theme handling
└── styles/         # Styling

🚀 Run Locally
# npm install
# npm run dev

📦 Production Build
# npm run build
# npm run preview
